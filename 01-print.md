# Printing

## Quick Reference

```python
# Print only a new line
print()

# Print a simple string message, followed by a new line
print('Hello World!')

# Print a string message with no newline at the end
print('No newline after this', end='')

# Print a string message with a space at the end
print('Put a space after this', end=' ')

# Print a string multiple times on the same line
print('%'*10, end='')

# Print a variable
my_var = 10
print(my_var)

# Print a variable combined with a string
my_var = 10
print('The value of the variable is:', my_var)

# OR

my_var = 10
print(f'The value of the variable is: {my_var}')

# Print two variables combined with a string
my_var = 10
my_second_var = 20
print('The value of variable one is', my_var, 'and the value of variable two is', my_second_var)

# OR

my_var = 10
my_second_var = 20
print(f'The value of variable one is {my_var} and the value of variable two is {my_second_var}')
```
